OpenAPI bindings

OpenAPI bindings for the file-diff-engine HTTP service. This contract covers repository discovery, job orchestration, file download, tokenization, and diff operations.

OpenAPI 3.1.0 Example deployment base URL: https://your-host.example.com

Components

This service publishes a machine-readable contract at ./openapi.json and a generated static website view built from the same source document.

Tags

NameDescription
systemService health and build metadata endpoints.
discoveryResolve refs, pull requests, and repository metadata before creating jobs.
jobsCreate jobs and inspect processed repository metadata.
filesDownload files and derive secondary artifacts from processed file blobs.
